# 系统架构设计师-法律法规与标准化 真题核心总结
本模块为**必考选择题（2-3分）**，考点高度固定，无计算题，纯**概念辨析、归属判定、期限记忆**三类题型，核心围绕**著作权、专利权、商标权、商业秘密**四大知识产权展开。

## 一、核心必背考点（速记版）
### 1. 保护期限（最高频）
1. **软件著作权**：**开发完成之日自动产生**，无需登记/发表
2. 永久保护（人身权）：**署名权、修改权、保护作品完整权**
3. 有限保护：发表权、财产权（复制/发行）→ 自然人终生+死后50年；法人发表后50年
4. 国家秘密期限：绝密30年、**机密20年**、秘密10年
5. 外观设计专利：相似设计最多**10项**

### 2. 保护对象与排除项
1. 软件著作权保护：**源程序、目标程序、配套文档**
2. 不保护：**思想、处理过程、算法、数学概念**
3. 不受著作权保护：法律法规、官方文件/译文
4. 商标权保护：**注册商标**（未使用/未注册不保护）
5. 专利客体：发明、实用新型、**外观设计（三者并列，无包含关系）**

### 3. 著作权归属（必考判定）
1. **委托开发**：无合同约定→**归创作方（受托方）**
2. **职务作品**：本职工作开发→**归单位**；离职**1年内**相关作品仍归原单位
3. 改编作品：著作权归**改编人**（不得侵犯原作）
4. 合作作品：归**实际创作人**（组织者无著作权）
5. 美术/摄影作品：原件转让仅转移**所有权+展览权**，**著作权不变**

### 4. 知识产权分类与规则
1. 知识产权归属：**民法范畴**
2. 工业产权：**专利+商标**；著作权≠工业产权
3. 专利许可类型：独占、排他、普通（**无特殊许可**）
4. 专利保护范围：**权利要求书**界定；专利必须申请授权才生效
5. 商业秘密：未公开、有价值、已保密；**不对抗善意第三人**
6. 合理使用：学习研究软件→**无需授权、无需付费、无需告知**

### 5. 侵权与免责判定
1. 盗版免责：**不知情持有者无责**，提供者无法证明合法来源→全责
2. 侵权行为：擅自修改/署名/复制发行、盗用职务作品、委托作品二次复用
3. 不侵权：个人学习研究、合法引用已发表作品（禁止引用未发表作品）

## 二、高频题型与秒杀技巧
1. **期限题**：记永久3项、50年著作权、20年机密、10项外观设计
2. **归属题**：委托归创作、职务归单位、原件不卖著作权
3. **侵权题**：未授权商用/署名必侵权；学习研究必免责
4. **概念题**：专利三类并列、商标只保护注册、商业秘密是知识产权客体

## 三、易混易错点（避坑必看）
1. 著作权**完成即生效**，登记仅为举证，非生效条件
2. 发表权**有期限**，≠永久保护的人身权
3. 职务作品开发者仅有**署名权**，无著作权
4. 专利权**必须申请**，与著作权自动生效相反
5. 引用未发表作品=侵权，仅可引用已发表作品
