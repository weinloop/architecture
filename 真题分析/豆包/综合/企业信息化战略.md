# 系统架构设计师-企业信息化战略 核心考点总结
本模块为**必考选择题（2~3分）**，**纯概念记忆、无计算**，考点高度固化，核心围绕**规划方法、转型阶段、集成架构、ERP、BI、电子政务、数据资产**七大方向。

# 一、战略规划与数字化转型
1. **企业数字化转型5阶段（必考顺序）**
初始级 → 单元级 → 流程级 → 网络级 → 生态级
2. **三大信息系统规划方法**
| 方法 | 核心特点 |
| ---- | -------- |
| CSF（关键成功因素法） | 识别关键因素，确定系统开发**优先次序**，分解至性能指标 |
| SST（战略目标集转化法） | 组织战略→信息系统战略，把规划视为**信息集合** |
| BSP（企业系统规划法） | 自上而下识别目标/流程/数据，**自下而上设计系统** |
3. **信息化需求三层次（高频）**
**战略需求（顶层）→ 运作需求（中层）→ 技术需求（底层）**
- 战略需求：提升竞争力、支撑长期发展
- 运作需求：运营策略、人才培养、流程落地
- 技术需求：系统升级、集成、性能安全

# 二、核心基础概念
1. **数据资产四大特征**
**可增值、可共享、可控制、可量化**（排除：可测试、可维护）
2. **信息化主体**：全社会（政府/企业/个人），**不是程序员/项目经理**
3. **霍尔三维结构（系统工程）**
- 三维：**时间维、逻辑维、知识维**
- 核心目标：最优规划、最优设计、最优管理、最优控制
- 研制阶段：输出**研制方案+生产计划**
4. **JavaEE遗产系统集成技术**：**JCA、JMS、Java IDL**（排除JDBC）

# 三、企业应用集成（EAI）高频考点
1. **EAI四层架构（从下至上）**
通讯服务 → 信息传递与转化服务 → 应用连接服务 → 流程控制服务
2. **企业内部集成层级（作用区分）**
- 应用系统集成：**系统互操作、数据/方法共享**
- 业务过程集成：**跨系统流程协同、信息共享**
3. **数据集成三种模式**
**数据联邦、数据复制、基于接口的数据集成**
4. **数据联邦核心**：虚拟统一视图，**各系统独立维护数据**，无集中存储
5. **事件驱动架构（EDA）**：模块**独立、非耦合**，基于消息代理异步通信

# 四、ERP（企业资源计划）
1. **ERP三流合一**：**物流、资金流、信息流**
2. **ERP五大计划（核心对应）**
- 生产计划大纲：经营计划细化
- 主生产计划（MPS）：**生产什么/多少/何时交**，ERP核心工作
- 能力需求计划（CRP）：**识别产能瓶颈、保障生产能力**
3. **核心模块**：库存管理 → 物料**进、出、存**管理

# 五、商业智能（BI）
1. **BI四大处理阶段**
数据预处理（**ETL：抽取/转换/加载**）→ 建数据仓库 → 数据分析 → 数据展现
2. **BI两大核心分析技术**
- OLAP（联机分析处理）：**多维分析**（上卷、下钻、旋转）
- 数据挖掘：**发现隐藏规律、预测未来**（分类、聚类、关联规则）
3. **数据挖掘核心任务**：聚类分析、分类分析、**关联规则挖掘**

# 六、电子政务（必考分类）
1. **三大行为主体**：政府、企事业单位、**公民（居民）**
2. **四大标准模式（排除G2C客户）**
- G2G（政府对政府）：人口信息采集、政务协同
- G2B（政府对企业）：营业执照颁发、招投标
- G2C（政府对公民）：户籍管理、社保服务
- G2E（政府对公务员）：内部人事、办公系统
3. 易错区分：企业投标 → **B2G**；人口信息采集 → **G2G**

# 七、电子商务与供应链
1. **电商核心实体**：客户、商户、银行、**认证中心（CA）**
2. **供应链信息流**
- 需求信息流：生产计划、客户订单
- 供应信息流：完工报告单、库存记录、入库单
3. **EDI四大模块**：通信、格式转换、联系、**消息生成与处理**

# 八、人工智能与专家系统
1. **专家系统核心**：**知识库 + 推理机**（设计中心）
2. **推理机作用**：规则匹配、逻辑推理、输出结论（系统思维核心）
3. **机器学习分类（易混）**
- 监督学习：**完全依赖标注数据**
- 半监督学习：**少量标注+大量未标注数据**
- 自监督学习：无人工标注，构造伪标签

# 九、高频易错秒杀清单
1. 电子政务无**G2C（客户）**，正确为**公民Citizen**
2. 数据资产**无**可测试、可维护；集成**无**数据转换/融合
3. ERP产能瓶颈→能力需求计划；生产明细→主生产计划
4. OLAP=多维查询；数据挖掘=预测/找规律
5. 企业集成底层：网络集成；顶层：接口集成
