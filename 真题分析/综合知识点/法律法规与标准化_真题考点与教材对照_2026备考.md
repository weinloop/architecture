## 一、真题 PDF 结构（小节划分）

| 小节 | 主要内容 |
|------|----------|
| **保护期限** | 软件著作权产生时间（**开发完成之日**）；人身权中**永久保护**（署名权、修改权、保护作品完整权等表述因题而异）；**发表权**常考「有期限」；法人/自然人软件**财产权保护期**（如 **50 年** 等典型数字） |
| **保护范围和对象** | 《软件产品管理办法》禁止情形；**商标同日申请**处理；著作权保护**表达**不保护思想（处理过程、思想方法）；**不适用著作权**的对象（**法律、法规、国家机关决议**等）；**合理使用**（引用限于**已发表**作品）；**软件注册商标**与商标权客体 |
| **产权归属** | **委托开发**无约定时归**受托方**；**职务软件**归单位；**合作/改编**著作权归属；**善意持有人**与**提供者**责任；**美术作品原件**所有权与**展览权**；盗版**不知情持有者**责任辨析；**商业秘密**因公开而丧失；**离职 1 年内**与原任务相关成果（常参照职务发明规则）；侵犯**复制发行**与商标侵权区分 |
| **知识产权（综合）** | **专利类型**（发明、实用新型、外观设计**并列**）；外观设计**相似设计数量上限**（真题解析称**10**）；**保密法**密级与**保密期限**（绝密/机密/秘密与年数）；专利**许可类型**（独占、排他、普通）；**权利要求书**确定保护范围；**学习研究软件**合理使用（条例第十七条）；软著登记**鉴别材料**（**源代码+文档**）；知识产权属**民法**范畴；**工业产权**（**专利、商标**等）；**商业秘密**作为知识产权**客体** |

> **说明**：本 PDF 文本中**单独成体系的「标准化」题目很少**（仅见《软件产品管理办法》中与**标准规范**相关的选项）。综合知识科目仍可能考 **国家标准编号、标准化法、常见 IT 标准角色** 等，建议用考纲+历年题补充，勿只依赖本题集。

|--------|----------------------|

| **不适用著作权** | **民航规章、规范性文件、官方文本** 等选项若考「是否作品」，按教材列举 **不适用** 情形判断 |
| **委托/职务归属** | **航司委托乙方** 开发平台，**无约定** → 常考归 **受托方**；**员工在职** 为 **单位任务** 开发的 **核心运行模块** → **职务作品** 归单位 |
| **侵权与善意持有** | **盗版光盘装离港插件**：**善意持有者** 与 **提供者** 责任不同；买正版 **许可范围外复制** 仍可能侵权 |

| **保密期限、密级** | **涉密运行数据、安保方案** 若题干涉密，按 **秘密/机密/绝密** 与 **年限** 选项作答（**以现行法与真题为准**） |
| **标准化** | 平台若对接 **行业接口、等级保护、机房与数据标准**，题干可出现 **GB/T、强制性/推荐性**（**以《标准化法》与大纲为准**） |

**【平台案例总览 · 航空运营智能管理平台】**

## 二、高频考点清单（便于背诵）

### 1. 著作权（含计算机软件）

- 软件著作权自**软件开发完成之日**自动产生；**登记非权利产生要件**。  
- 受保护对象：**源程序、目标程序、文档**；一般不保护**思想、处理过程、操作方法、数学概念**等。  
- 人身权与财产权区分：**署名权、修改权、保护作品完整权**——真题常考「不受期限限制」；**发表权**——多题强调「有期限」或与著作权保护期关联。  
- **合理使用**：课堂教学、学术研究等对软件**设计思想与原理**的安装使用（《计算机软件保护条例》相关条）；**学术论文引用**须引用**已发表**作品。  
- **不适用著作权法**：法律、法规，国家机关的立法/司法/行政性质的文件及其**官方正式译文**等（选项常给《计算机软件保护条例》作例子）。

**【平台案例】**  
- **开发完成之日**：可记「**最后一个模块联调通过可独立运行**」类 **完成节点**（具体仍以题干与法律表述为准）。  
- **思想 vs 表达**：**「旅客画像策略」** 与 **「具体算法代码与界面」** 区分选项中的 **不保护对象**。  
- **合理使用**：**院校** 用正版软件 **教学**、**引用已发表论文** 等典型 **合理使用** 叙述。

### 2. 商标与专利（选择题常混考）

- **注册商标**受商标法保护；侵犯软件复制发行多为**著作权**而非商标专用权（视题干而定）。  
- 专利**须申请授权**；**发明、实用新型、外观设计**为三类客体；**实用新型不包含外观设计**。  
- **权利要求书**界定专利保护范围；说明书支撑权利要求。  
- 专利许可类型：**独占、排他、普通**（无「特殊许可」为正式类型之说）。  
- **工业产权**：典型为**专利、商标**（著作权归入版权类，不属工业产权）。

**【平台案例】**  
- **三类专利**：**新机载支架结构** 可能涉及 **发明/实用新型**；**App 图标** 常考 **外观设计** 与 **商标** 分题。  
- **权利要求书**：**行李分拣专利** 若题干问 **保护范围**，选 **权利要求书** 相关表述。  
- **许可类型**：**独占/排他/普通** 与 **某航司独家使用某专利引擎** 类题干对应。

### 3. 侵权责任与归属

- **委托作品**：无书面约定时，著作权归**受托人**（完成者）。  
- **职务软件**：通常归**单位**；登记名义与真实归属不一致时，以**法律规定**为准。  
- **侵权复制品**：持有者**善意**可免责；**提供者**不能证明**合法来源**须担责。  
- 买正版后仅享有**许可范围内使用权**，**不当然享有复制权**。

**【平台案例】**  
- **委托开发**：**航司** 委托 **集成商** 建 **AODB 接口模块**，合同未约定著作权 → 记 **受托方** 高频选项。  
- **职务软件**：**信息中心员工** 编写 **配载子系统**，属 **本职工作** → 记 **法人/单位** 归属。  
- **善意持有**：**航站** 批量安装 **来源不明补丁** 的责任辨析，跟题干 **「是否善意、是否提供者」** 走。

### 4. 其他法律条（真题已出现）

- **国家秘密保密期限**（除另有规定外）：**秘密≤10 年，机密≤20 年，绝密≤30 年**（以现行《保守国家秘密法》及真题解析为准，修订后若有调整以最新法条为准）。  
- **知识产权部门法属性**：通说为**民事特别法**，选项常考「属于**民法**范畴」。

**【平台案例】**  
- **保密期限**：涉及 **运行数据出境、安保等级评估** 的题干若出现 **绝密/机密/秘密** 与 **年数**，按 **真题与现行《保守国家秘密法》** 数字选，**勿用业务直觉替代法条**。  
- **民法范畴**：选项问 **知识产权法与民法关系** 时，按教材/通说 **民事特别法** 作答。

## 三、与《系统架构设计师教程》的衔接关系

| 教程位置 | 可衔接内容 |
|----------|------------|
| **第2章 2.3.1 计算机软件概述** | 软件 = **程序 + 文档**；说明文档是软件组成部分——与著作权法保护「作品」表达形式一致，可作概念入口。 |
| **第4章 信息安全**（管理体系中的**法律管理**） | 信息系统安全需符合**法律法规**——与「合规、侵权责任」类题目态度一致，但**无具体法条**。 |
| **全书** | **无**与真题 PDF 同等粒度的著作权法、专利法、商标法条文讲解。 |

**结论**：该主题应以**《著作权法》《计算机软件保护条例》《专利法》《商标法》**等条文及官方教材/考纲辅导为主；架构师教程仅作**背景概念**辅助。

**【平台案例】**  
- 教程中 **软件 = 程序 + 文档**（第2章）可与 **软著登记鉴别材料** 一起记：**源代码 + 文档** 都是 **表达载体**。  
- **信息安全法律管理**（第4章脉络）可与 **等保、数据合规** 类选项 **态度一致**：系统建设需 **依法合规**——但 **具体题** 仍回到 **著作权/保密/标准化** 条文。

## 四、2026 年 5 月考查倾向预测（供参考）

1. **著作权产生时间、职务/委托归属、侵权认定**仍为送分与拉分混合区，选项干扰性强，需抠准「**完成日**」「**受托人**」「**单位任务**」。  
2. **专利与商标、著作权**三元区分（权利取得方式、保护对象、是否申请）继续适合出概念题。  
3. **新法修订点**：若考试周期内著作权法、专利法实施细则、保密法等**修订生效**，注意**保护期、合理使用、外观设计**等数字与表述更新。  
4. **标准化**：本题集覆盖偏少，建议单独记忆 **GB/T、GB/Z、行业标准与国家标准关系**、**强制性标准与推荐性标准** 等通用考点（以最新《标准化法》及大纲为准）。  
5. **案例助记**：**航空运营智能管理平台** 仅用于 **委外、职务、三类知识产权、保密数字** 的 **情景联想**；**新法修订** 以当年 **大纲与真题解析** 为准，不在业务故事里硬记年限。

## 五、复习建议（简要）

1. 用本 PDF 的**四小节**做框架，把错题对应到「期限—客体—归属—综合」四类笔记本。  
2. **数字题**（50 年、10 项相似外观、保密年限）单独整理一页，考前速览。  
3. 教程不必强行对应章节；时间紧时**直接刷真题+官方条文**效率更高。  
4. 一张 **「平台一句话」**：**合同没写找受托，上班写的找单位，思想不保护表达保，保密数字看真题法条。**
