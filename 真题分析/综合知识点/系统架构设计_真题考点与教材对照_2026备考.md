## 一、常考 · 直接背诵

### 1.1 架构风格（§7.3 + PDF 最高频）

**定义句**：风格 = **词汇表 + 约束**；重用 **设计** 层模式。

**大类与关键词**（与《04》四一致，此处一句版）

| 类 | 记区别 |
|----|--------|
| **数据流** | **批处理** 整批步序 vs **管道-过滤器** 流式、可流水并行 |
| **调用/返回** | 主程序/OO/**层次**/ **C/S**（二层/三层：**应用服务器**） |
| **数据中心** | **仓库** 中央数据 + 独立构件；**黑板** **分级解空间**、多知识源 |
| **虚拟机** | **解释器**（脚本/自定义）；**规则系统**（规则+解释器+**工作内存**） |
| **独立构件** | **进程通信**（显式消息/RPC）vs **事件/隐式调用** |
| **C2** | 构件 **不直连**，经 **连接件** **顶/底** 规则连接 |

**场景速配（真题变体）**：规章常变 → **规则系统**；用户脚本 → **解释器**；编译多阶段 → **管道**/批处理；IDE 共享 AST → **仓库**；代码改动多工具响应 → **事件**；语音多源 → **黑板**；分层脆弱性 → **底层故障/层间开销波及上层**。

---

### 1.2 ABSD / ABSDM（§7.2 + 《04》三）

- **驱动**：**商业 + 质量 + 功能**。  
- **三基础**：**功能分解**、**选风格**、**软件模板**。  
- **六子过程**（顺序题）：**许设温敷实验** —— 需求 → 设计 → **文档化** → **复审** → **实现** → **演化**（**无**「体系结构测试/管理」标准干扰项）。  
- **捕获**：**用例** + **质量场景**（变更/性能/可靠性/交互性，含非预期）。

---

### 1.3 ADL · 多视图 · 失配（§7.1.2）

- **ADL 突出**：**构件、连接子、配置**（接口等）。  
- **多视图**：关注点分离 —— **4+1** 见下节。  
- **构件组装失配**（Garlan 等）：**假设不一致**（对基础设施/协议/数据模型等）— **按题选项**。  
- **中间件**：可实现 **连接子**，衍生 **中间件导向风格**。

### 1.5 DSSA（§7.5 + 《04》五）

- **垂直域** / **水平域**。  
- **三活动**：领域分析 → 领域设计（**DSSA**）→ 领域实现。  
- **四角色**：领域专家、分析、设计、**实现**。  
- **五阶段**（Tracz）：范围 → 领域特定元素 → 设计/实现约束 → 领域模型与结构 → **可重用产品单元**。

---

### 1.6 复用与产品线（§7.4 + 《04》五）

- **产品线**：**核心资产库** + 规定集成方式。  
- **机会复用** vs **系统复用**（规划驱动）。  
- **三阶段**：获取 → **管理**（构件库）→ **使用**（选、定制、组装）。

---

### 1.7 架构评估与质量（**第8章** + 《04》六）

**九大属性、六要素、敏感/权衡** —— **全表背《04》六**。

**补充（PDF 高频）**：

- **SAAM**：输入常考 **问题描述、需求声明、架构描述**；场景 **刺激、环境、响应**。  
- **ATAM**：**四质量**（常考 **性能、可用性、安全性、可修改性** —— **以§8.3教材为准**）；**效用树**：根 → 质量属性 → 分类 → **场景**；活动含 **视图与场景实现、属性模型、折中**；产出 **敏感点、权衡点**。  
- **CBAM**：**ROI**、效用与成本。  
- **SASAM**（部分真题）：**静态** 架构分析（2023 等）。

## 二、下次考试预测 · 直接背诵

**高概率**

1. **批处理 vs 管道**；**仓库 vs 黑板**；**规则 vs 解释器 vs 事件**。  
2. **三层 C/S** 各层；**层次脆弱性**。  
3. **ABSD 六步顺序**、三基础、驱动因素。  
4. **DSSA** 垂直/水平、三活动、**四角色名**、五阶段。  
5. **敏感点 vs 权衡点**；**效用树** 结构。  
6. **ATAM** 活动；**质量场景六要素**。  
7. **失配**、**ADL 三要素**。  

**中概率**

8. **4+1** 职责；**SAAM** 输入。  
9. **产品线**、**机会/系统复用**。  
10. **SOA vs 微服务**。  
11. **CBAM ROI**。  

**低概率**

12. **SASAM**；**胖瘦客户机** 细辨。  

**预测口诀**

**批整步序管流式管道仓中央黑板分级虚拟机解释规则工内存事件隐式C2顶底不直连层次脆弱底病上层蔓延，ABSD商质功三分解风格模板许设温敷实验六步无测管，ADL构件连接子配置失配假设多视图4+1罗凯进五常，DSSA垂族水片四专家三析设实五阶范围到资产，复用机会系统获取管选用产品线资产库，SAAM三输入场景刺激回应ATAM效用树四质量折中敏感将影响权衡升降档CBAM钱效，SOA粗总线微细小去中心云原题干认。**
