## 一、常考 · 直接背诵

### 1.1 第2章 · OS 定位与组成（必背）

- **第一层软件**；**处理机、存储、设备、文件** 资源管理。  
- **内核**：进程/任务、存储、文件、设备；**抽象硬件**；**系统调用 / API** 给应用。  
- **访管中断 / 陷入**：用户态 → 内核态（题干常考）。

---

### 1.2 四大特征

| 特征 | 一句 |
|------|------|
| **并发** | 单 CPU **宏观并行、微观串行**；多 CPU 真并行 |
| **共享** | **同时共享** 与 **互斥共享** |
| **虚拟** | 物理 ↔ 逻辑映射（虚存、虚拟机等） |
| **异步（不确定）** | 推进 **次序与速度不可预知** |

---

### 1.3 七大类 OS · 关键词（教材列举）

**批处理**（单道/多道；多道：**宏观并行、微观串行**、提高利用率）  
**分时**（**时间片**；多路性、独立性、**交互性**、及时性）— **时间片固定、用户增多 → 轮转周期变长、响应变差**  
**实时**（**被控对象允许时间内**响应；实时**控制** vs 实时**信息处理**）  
**网络 OS** | **分布式 OS** | **微型机 OS** | **嵌入式 OS**

辨析：**旅客交互** → 常 **分时**；**报表 T+1** → **批处理**；**防撞/关键联锁** → **硬实时**。

---

### 1.4 嵌入式操作系统（第2章 + 真题扩展）

**特点（背）**：**微型化、可定制、实时性、可靠性、易移植性**（**HAL、BSP**）。  
**典型产品（教材表）**：**VxWorks、μClinux、PalmOS、Windows CE、μC/OS-Ⅱ、eCos** 等。

**扩展高频词**：**鸿蒙** 微内核；RTOS **可裁剪**、**抢占**；**看门狗**；**JTAG**；**低功耗**（编译/软硬件协同/算法）；**混成**、**EDF/RMS/LLF** 等调度名 —— **随题**。

---

### 1.5 原理速记块（与《11》一致，压缩成条）

**进程 · 三态 / 五态**

- **三态**：《11》口诀 **舅运祖宗** — **就绪 / 运行 / 阻塞**；**阻塞不可直接回运行**（须经就绪）。  
- **五态**：新 → 就 → 运 → 阻 → 终（**新舅运祖宗**+终）。

**PV · 信号量**

- **P**：S−1；S<0 **阻塞**，**|S|** 为等待数。  
- **V**：S+1；S≤0 **唤醒一个**。  
- **互斥** 初值常 **1**；**前趋图**：**前驱完成 V、后继开始 P**，**每条边** 常设一信号量 **初值 0**。

**死锁**

- **四条件**：互斥、不剥夺、请求与保持、**环路**。  
- **预防**：破坏任一条件。  
- **同类资源下界**：**n 进程各需 r 个** → 至少 **n(r−1)+1** 才可能避免死锁（**按题算**）。

**页 / 段 / 置换**

- **页**：逻辑 = **页号 + 页内偏移**；物理 = **块号×页长 + 页内**。  
- **段**：逻辑 = **段号 + 段内**；物理 = **基址 + 段内**；**越界** 段式常考。  
- **置换**：**FIFO / LRU / Clock**；**访问位·修改位** 参与 Clock/LRU 变体。

**文件 · 索引 · 位示图**

- **打开文件**：先读 **目录项 / FCB（元数据）**；**绝对路径** 自 **根** 起。  
- **inode 直接/一级/二级间接**：逻辑块号分段 —— **按《11》例题公式**。  
- **位示图字数** = **物理块总数 / 字长**。

**磁盘 · 调度 · 缓冲**

- **SSTF** 等移臂；**双缓冲** 时间线 —— **按题算**。

## 二、下次考试预测 · 直接背诵

**高概率**

1. **第2章原文**：分时 **时间片与响应**；实时 **时限**；嵌入式 **特点+产品名+ BSP/HAL**。  
2. **PV + 前趋图** 填空；**S=-3** 问 **阻塞进程数**。  
3. **银行家** 安全序列；**资源下界** 公式。  
4. **进程三态** 哪条转换 **非法**。  
5. **死锁四条件** 与 **预防**。  
6. **分页** 逻辑→物理 **一位运算题**。  

**中概率**

7. **页面置换** 缺页次数 **模拟**；**FIFO Belady**。  
8. **inode / 索引** 块号落在哪级间接。  
9. **位示图** 字数。  
10. **SSTF**、**双缓冲**。  
11. **TCP/设备** 若混在 PDF 则归网络/设备子题。  

**低概率 / 防偏**

12. **契约式设计 DbC** 等混排 —— **不归 OS 则按软件工程**。  

**预测口诀**

**并发共享虚拟异步分时片长实时限，嵌入式Vx裁剪BSP抢占狗，PV前趋P减阻塞绝对值死锁四路破一环，银行安全下界n倍r减一加一，页号页内块乘加段基越界FIFO LRU Clock缺页数，打开先FCB根路径位示块除字长SSTF双缓冲算时间。**

## 七、考前一页纸

1. **四大性**：并发、共享、虚拟、**异步**。  
2. **分时**：时间片；用户多 → **周期变长**。  
3. **实时**：**允许时间内**；嵌入式：**微型可裁剪实时可靠移植 + BSP/HAL**。  
4. **三态**：就绪 / 运行 / 阻塞；**阻塞不经就绪不可直达运行**。  
5. **PV**：P−1 负则等；V+1 非正则唤醒；**|S|** = 阻塞数。  
6. **死锁**：四条件；**n(r−1)+1** 下界。  
7. **页**：页号+页内 → 块号拼接。  
8. **置换**：FIFO/LRU/Clock。  
9. **位示图**：块数÷字长。  
10. **详算细节** → **`选择/11-操作系统-考点速记.md`** 各节例题思路。
