## 一、常考 · 直接背诵

### 1.1 MVC 与 Web 分层（§13.2.1 / 第12章）

**MVC 职责**

- **Model**：业务状态与规则。  
- **View**：展示、少业务判断。  
- **Controller**：接请求 → 调 Model → **选视图 / 状态查询**。  

**SSM 链（2020 等）**

- 浏览器 → **Spring MVC（Controller）** → **Service** → **MyBatis Mapper** → **连接池** → DB。

**与 J2EE（2016-17）**

- **JSP/Servlet**、**JavaBean**、**Service**、**DAO**；或 **EJB** 会话/实体等 —— **随空填**。

---

### 1.2 SOA 与 ESB

- **SOA**：**封装、松耦合、可复用** 服务。  
- **ESB**：**路由**、**协议/格式转换**、**中介**、**安全/监控**、服务位置透明；**UDDI/publish** 类比 **服务目录**。  
- **HTTPS** + **防火墙** + **摘要/签名**（2018 类）。

---

### 1.3 质量属性 · 非功能归类（2019、2020）

| 类型 | 关键词 |
|------|--------|
| **性能** | 时延、吞吐、并发、峰值 |
| **安全** | SQL 注入、XSS、审计、加密 |
| **可用性** | 7×24、**RTO/RPO**、双活 |
| **易用性** | 学习时间、少步操作、无障碍 |

**口诀**：**秒级并发→性能**；**学会多久→易用** —— 勿混。

---

### 1.4 CDN、反向代理、读写分离（§10.6.6）

- **CDN**：静态资源 **边缘缓存**，减源站。  
- **反向代理**：**TLS 终结、限流、缓存**，隐藏后端集群。  
- **主从 + 读写分离**：读从、写主 —— 与 Redis 等 **缓存** 叠用（见数据库文档）。

---

### 1.5 TCP vs UDP（第4章）

| | TCP | UDP |
|---|-----|-----|
| **可靠性** | **面向连接、可靠、有序** | **无连接、不保证可靠** |
| **场景** | 支付、离港、控制面 | 可容忍丢包、低延迟（按题干） |

---

### 1.6 MQTT 与边缘（2022、2021）

- **MQTT**：**发布/订阅**、**Broker**、**轻量**、设备侧；**QoS** 按必达性选。  
- **边缘计算**：**本地预处理、降延迟、省带宽、隐私**；与 **中心云** 协同。

---

### 1.7 数据访问与 ORM（2015、2017-4、2023）

- **ORM**：对象-表映射；**移植** 换库少改应用代码。  
- **Hibernate**：**配置 / 映射（XML 注解）/ JDBC**.  
- **MyBatis**：SQL Mapping。  
- **DAL + 抽象工厂**：**异构库** 选不同 **DAO 实现**。  
- **SQL 注入**：**参数化** + 输入校验。

---

### 1.8 JWT（2023）

- 三部分：**Header、Payload、Signature**。  
- **Payload 勿塞敏感明文**；**HTTPS**；**过期与吊销**。

---

### 1.9 Redis / ES / 缓存（2023～2025，与数据库篇交叉）

- **Redis 主从**：**PSYNC/SYNC**、**BGSAVE**、**RDB 传输**、**backlog**；**RDB vs AOF** —— **按真题填空**。  
- **分布式锁 / 击穿**：**Cache-Aside**；**物理过期+互斥锁** vs **逻辑过期** —— **2025-11**。  
- **ZSet 秒杀**：**ZADD、ZRANGEBYSCORE、ZSCORE** 等 —— **2024-05**。  
- **ES 分词**：**Simple / Whitespace / Keyword**（精确不分词）— **2024-11**。  
- **DB 实现分布式锁缺点** vs **Redis 锁死锁** —— **2024-05**。

---

### 1.10 端侧 AI vs 云端 AI（2025-05）

- **端侧**：低延迟、隐私、**边缘推理**；受算力约束。  
- **云端**：**大模型、集中调度**、GPU 池。  
- **资源池**：**抽象、调度、隔离**；集中式缺陷：**单点瓶颈、 noisy neighbor、运维复杂度** —— **随题展开**。

---

### 1.11 其他真题词

- **响应式 Web**：**流式布局**、**媒体查询**（2017-5）。  
- **扩展接口 + 策略 + XML**（2014-1、2013）：表现层 **可配置、插件化** —— **按题干**。  
- **DNS vs 反向代理** 负载均衡；**Memcached vs MySQL query cache**（2014-5）。  
- **PHP 局限 vs J2EE**；应用服务器 **集群、连接池、事务**（2016）。

## 二、下次考试预测 · 直接背诵

**高概率**

1. **MVC + 调用链**（SSM / Servlet 体系）填空。  
2. **SOA + ESB** 能力 + **服务注册/发布** 类比词。  
3. **非功能四类** 归类 + **易用≠性能**。  
4. **CDN / 反向代理 / 负载** 与 **§10.6.6** 话术。  
5. **TCP/UDP** 选一题；**MQTT + 边云协同** 一题。  
6. **JWT 三部分**；**Redis/AOF/RDB/主从**（与数据库篇同练）。  
7. **Cache-Aside + 击穿治理**（序列图级）。  

**中概率**

8. **ES Keyword**；**推荐系统分层**；**REST + 前后端分离**。  
9. **ORM vs 手写 SQL**；**工厂 + DAL**。  
10. **端侧 vs 云端 AI** + **资源池三要素**。  

**低概率 / 防偏**

11. **扩展接口模式**、**XML 表现层**Vintage。  
12. **DNS 负载** 与 **七层反代** 辨析。  

**预测口诀**

**MVC链SSM离港库，SOA封装ESB转路由，四类非功易用别归吞吐，CDN边反代TCP可靠UDP飘，MQTT边云JWT三段Redis主从AOF敲，击穿互斥逻辑过期或Cache旁，ES Keyword端云池隔离调度要记牢。**

## 四、与《04-云原生架构设计·考点速记》交叉 · 直接背诵

**七原则（每条背一词）**

| 原则 | 关键词 |
|------|--------|
| **服务化** | 拆服务、治理流量 |
| **弹性** | **资源随业务扩缩**（**≠** 韧性） |
| **可观测** | 日志、链路、指标 |
| **韧性** | **故障下** 熔断、降级、多 AZ |
| **全流程自动化** | CI/CD、灰度、回滚 |
| **零信任** | 身份、mTLS、网关鉴权 |
| **持续演进** | 特性开关、渐进发布 |

**弹性 vs 韧性 · 口诀**：《04》— **弹性对业务量，韧性对故障**。

**七模式（各一句）**

- **服务化**（微服务/SOA）；**Mesh**（Sidecar 统一治理）；**Serverless**（事件触发、免管服务器）；**存算分离**（无状态+分布式存储）；**分布式事务**（XA、消息最终一致、TCC、SAGA 等 —— **选型随题**）；**可观测**（Logging/Tracing/Metrics）；**EDA**（Kafka 等事件解耦）。

**反模式三句**

- **庞大单体** → **适度拆分**；**硬拆微服务** → **粒度与组织匹配**；**缺自动化** → **流水线+IaC**。

**容器 vs VM（对比背）**

| | 容器 | VM |
|---|------|-----|
| 内核 | **共享宿主机内核** | **独立 Guest OS** |
| 启动 | **秒级/毫秒级** | **分钟级** |
| 隔离 | **较弱** | **强** |

**容器 vs 编排**

- **Docker** = **单容器** 打包运行；**Kubernetes** = **集群** 调度、伸缩、发现、恢复 —— **口诀**：容器管一盒，编排管一簇。

**Docker / kubectl 高频词**：`docker ps`、`run`、`logs`、`build`；`kubectl get pods`、`describe`、`logs` —— **《04》表**，案例 **偶考指令**。

**与 Web 真题合写一句**

- **Ingress/Nginx 反代** + **k8s HPA** = **§10.6.6** 的「边缘」与《04》的 **弹性**；**JWT + mTLS** = Web **无状态** 与 **零信任** 合流。

## 七、考前一页纸

1. **MVC**：**模型—视图—控制器** 三职责；SSM **链** 背一遍。  
2. **ESB**：**路由、转换、中介、安全/监控**。  
3. **四类非功能**：性能 / 安全 / 可用 / **易用**。  
4. **CDN + 反代**：**静态边缘、动态反代护源**。  
5. **TCP 可靠**，UDP **不保证**；**MQTT 订阅轻量**。  
6. **JWT** 三段；**Redis 细节** 转 **数据库背诵版**。  
7. **弹性≠韧性**（《04》）；论文可 **七原则** 一句话带 **10 万并发/5000TPS**。  
8. **交叉文档**：缓存/GAV 与 **`数据库系统与缓存设计_…`** 同步，避免两套矛盾。
