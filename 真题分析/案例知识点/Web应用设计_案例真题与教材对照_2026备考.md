| 真题考点 | 教程依据 |

### 2. SOA 与 ESB（2018）

| 真题考点 | 教程依据 |
|----------|----------|
| **SOA**：服务封装、松耦合、异构集成 | **第12章「3. 面向服务架构」**；**§1.1.3** 架构风格应用场景 |
| **Web Service**、SOAP/WSDL 表述 | **第12章「2）Web Service」** |
| **ESB**：路由、协议/格式转换、中介、安全与监控 | **§13 章**及扩展阅读；真题图与 **企业服务总线** 标准表述一致 |

**【平台案例】**  
- **SOA**：将 **航班计划、旅客服务、资源管理** 封装为 **粗粒度服务**，供 **门户、App、第三方** 调用，**松耦合** 替换某 legacy 子系统。  
- **ESB**：**SOAP/REST、报文、时区** 不一，经 **ESB** 做 **路由、协议转换、格式映射**；**UDDI/publish** 类比 **服务目录**（**航司 API 网关 + 服务注册**）。  
- **HTTPS / 防火墙**：对外 **旅客与代理人** 通道 **TLS**；对内 **分区防火墙** 隔离 **运行网与办公网**（与 **2018** 安全填空同构）。

### 3. 质量属性与非功能需求归类（2019、2020）

| 真题考点 | 教程依据 |
|----------|----------|
| **性能**（吞吐、时延、实时性） | **§8.1** 质量属性场景 |
| **安全性**（XSS、SQL 注入、审计） | **§8.1**；**第4章** 安全相关 |
| **可用性**（7×24、**RTO** 恢复） | **§8.1 可用性** |
| **易用性**（界面友好、学习时间） | **§8.1**（或并入用户友好类场景） |

**【平台案例】**  
- **性能**：**早高峰** 并发查动态、**行李查询 QPS**；指标 **时延、吞吐、峰值**。  
- **安全**：**SQL 注入**（恶意查他人行程）、**XSS**（留言板）、**操作审计**（谁改了舱单）。  
- **可用性**：**运控双活**、**RTO**（席位恢复时间）；**易用性**：**老年旅客**、**柜台少步完成**、**无障碍**。  
- **归类口诀**：带 **「秒级、并发」** 多归 **性能**；带 **「学会要多久」** 归 **易用性**；勿把 **易用性** 误归 **性能**。

### 4. 高并发 Web 架构：CDN、反向代理、缓存、读写分离（2014、2017）

| 真题考点 | 教程依据 |
|----------|----------|
| **CDN 与反向代理**：缓存、加速、隐藏后端 | **§10.6.6 第六阶段：使用反向代理和 CDN 加速网站响应** |
| **负载均衡** 与调度 | **§10.6** 同章前后文（负载均衡调度图） |

**【平台案例】**  
- **CDN**：**旅客 App 静态资源**（JS/CSS/图片）、**机队手册 PDF** 边缘缓存，减轻 **源站** 压力。

### 5. 传输层与物联网协议（2021、2022）

| 真题考点 | 教程依据 |
|----------|----------|
| **TCP**：面向连接、可靠、有序、拥塞控制 | **第4章 §4.1～4.2**（计算机网络基础） |
| **UDP**：无连接、不保证可靠、低延迟 | 同上 |
| **MQTT**：发布/订阅、Broker、轻量、适用物联网 | **教程无专节**；以真题与扩展资料为准 |
| **边缘计算**：延迟、带宽、隐私、本地推理 | **第11章/新技术** 可能有边缘与云协同线索；答题以 **真题归纳** 为准 |

**【平台案例】**  
- **TCP**：**值机支付、离港提交、签派放行** 等 **强一致、不可丢** 的控制流用 **TCP**（**2021** 可靠性论述）。  
- **UDP**：**实时性优先、可容忍丢包** 的 **场面视频预览、部分广播**（若题干强调延迟）。  
- **MQTT**：**行李转盘 RFID、廊桥状态、除冰车位置** 等 **设备侧** 经 **Broker** 上报云端；**QoS** 按 **是否必达** 选型。  
- **边缘计算**：**航站本地** 做 **报文预处理、图像初判、断网缓存**，再与 **中心运控** 协同，降低 **带宽** 与 **延迟**（**2022** 边云协同优势）。

### 6. 数据访问与 ORM（2015、2017-4、2023）

| 真题考点 | 教程依据 |
|----------|----------|
| **持久层**、分层解耦 | **§13** 层次式架构 + 软件工程分层实践 |
| **ORM** 优缺点、移植性 | **案例表述**；教材 **§6.3** 侧重关系设计，ORM 作 **工程方法** |
| **抽象工厂 / 工厂方法** 在 DAL 创建不同 DB 访问对象 | **第2章 §2.x UML/设计模式** 或 **5.3.2** 面向对象设计（工厂模式） |
| **Hibernate** 配置 / 映射 / JDBC | **真题**；教程 **J2EE / 持久化** 分散提及 |

**【平台案例】**  
- **持久层**：隔离 **SQL 细节**，上层 **Service** 只操作 **领域对象**（航班、旅客订座记录）。  
- **ORM**：**航司主数据、机组排班** 等 **对象-表** 映射；**移植** 指换库时 **少改 Java**（仍要验 SQL 方言）。  
- **抽象工厂 + DAL**：不同航站接入 **异构离港/订座接口** 时，**工厂** 创建 **不同 DAL 实现**（**2017-4** 异构库同构）。  
- **Hibernate 体系**：**hibernate.cfg.xml**、**实体映射**、底层 **JDBC**（**2023** 填空领域化）。

## 三、高频默写清单

1. **MVC** 三角职责 + **图 13-2** 三条交互（选择视图 / 业务处理 / 状态查询）。  
2. **SOA** 与 **ESB** 一句话 + ESB 典型能力（路由、转换、中介、安全）。  
3. **质量属性四类**：性能、安全、可用、易用（含典型子指标）。  
4. **CDN vs 反向代理** 与「尽早返回」思路（§10.6.6）。  
5. **TCP vs UDP** 可靠性对比 + 控制类业务选 TCP。  
6. **MQTT** 发布/订阅 + 轻量 + 适用设备侧。  
7. **SQL 注入**：参数化查询 + 输入校验。  
8. **响应式 Web**：流式布局 + **媒体查询**（真题）。  
9. **SSM**：Spring + **Spring MVC** + **MyBatis** + **连接池** 调用链。

**【平台案例 · 与上表逐条对应】**  
1. MVC：**运控** 选视图 / 处理改航 / 查状态。  
2. SOA/ESB：**AODB/DCS** 经总线 **路由+转换**。  
3. 四类：**动态时延**、**舱单安全**、**双活**、**柜台易用**。  
4. CDN/反代：**App 静态** CDN；**API** 反代。  
5. TCP：**离港提交**；UDP：低延迟可丢场景（按题干）。  
6. MQTT：**转盘/廊桥** 设备上报。  
7. SQL 注入：**参数化** 查行程。  
8. 响应式：**旅客 App** 多屏。
