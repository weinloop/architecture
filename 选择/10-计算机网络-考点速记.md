## 二、传输方式与码元

- **单工**：数据只能单方向传输（如广播）。**半双工**：双方可收发但不能同时（如对讲机）。**全双工**：可同时双向（电话、以太网）。
- **码元**：代表不同离散值的基本波形；V 进制码元可携带 log2(V) 比特。

**案例（航空运营智能管理平台）**：旅客 **广播式** 接收航变公告近似 **单工**；对讲式客服信道为 **半双工**；**浏览器↔API**、**数据库主从同步** 常用 **全双工** 以太网。高阶调制提高 **每码元比特数**，与 **链路容量规划** 相关。

---

## 三、奈氏准则与香农公式（★★★★★）

- **奈氏准则**（理想低通、无噪声）：极限码元速率 = 2W（码元/秒）；**极限数据传输率 = 2W × log2(V)**（b/s），V=码元状态数。
- **香农公式**（有噪声）：**C = W × log2(1 + S/N)**（b/s）；信噪比 dB = 10×log10(S/N)。实际速率低于 C 才可能无差错。

**案例（航空运营智能管理平台）**：机房 **互联带宽、公网出口、跨地域复制** 可用 **香农** 理解噪声与误码对 **有效吞吐** 的上限；**奈氏** 帮助理解 **给定带宽与调制阶数** 下的 **码率上限**。春运峰值下 **“带宽够但仍卡顿”** 常与 **丢包、重传、队头阻塞** 有关，不单是公式层。

## 四、数据编码（了解）

- **模拟**：AM/ASK、FM/FSK、PM/PSK；抗干扰 PM 最强。
- **数字**：不归零（无同步）、**曼彻斯特**（每比特中间跳变，高→低=1）、差分曼彻斯特（中间跳变同步，起始无跳变=1）、4B/5B 等。

**案例（航空运营智能管理平台）**：考点以辨识为主；工程上关注 **PHY 以上**——**TLS 加密**、**HTTP 语义**、**TCP 重传** 对 **≤1 秒购票响应** 的影响。

## 五、OSI 七层与协议（★★★★★）

**口诀**：**物联网书会使用**——物、数、网、传、会、表、应。

| 层 | 名称 | 常见协议 |
|----|------|----------|
| 7 应用 | 为应用提供网络服务 | HTTP、FTP、SMTP、POP3、DNS、Telnet、SNMP |
| 6 表示 | 格式转换、加密压缩 | SSL/TLS |
| 5 会话 | 建立、管理、终止会话 | NetBIOS、RPC |
| 4 传输 | 端到端传输 | TCP（可靠、面向连接）、UDP（无连接） |
| 3 网络 | 路由与寻址 | IP、ICMP、IGMP、ARP、RARP |
| 2 数据链路 | 帧、差错、流量控制 | PPP、HDLC、Ethernet、VLAN、MAC |
| 1 物理 | 物理标准与信号 | RS-232、RJ-45、IEEE 802.3 |

**案例（航空运营智能管理平台）**：

- **应用层**：**HTTPS** 购票与看板、**DNS** 解析 API 域名、**SNMP**（若用）做网络监控。
- **表示层**：**TLS** 与素材 **SSL/TLS 全程加密**、敏感字段保护一致。
- **会话/传输**：**TCP** 保障 **支付回调、订单提交**；**UDP** 用于 **DNS、部分实时音视频/日志采集**（依实现）。
- **网络及以下**：**VPC 内路由、跨 AZ、负载均衡** 对应 **网络层与链路**；容器 **CNI** 在 **二/三层** 实现东西向流量。

## 六、TCP/IP 四层与常见端口

**口诀**：**应传网接**——应用、传输、网际、网络接口。

| 传输层 | 协议 | 端口 | 说明 |
|--------|------|------|------|
| TCP | FTP | 20 数据、21 控制 | 文件传输 |
| TCP | Telnet | 23 | 远程登录 |
| TCP | SMTP | 25 | 发邮件 |
| TCP | HTTP | 80 | Web |
| TCP | POP3 | 110 | 收邮件 |
| UDP | DNS | 53 | 域名解析（也可 TCP） |
| UDP | DHCP | 67/68 | 自动分配 IP |
| UDP | TFTP | 69 | 简单文件传输 |
| UDP | SNMP | 161/162 | 网络管理 |

**TCP**：可靠、面向连接、重发；**UDP**：无连接、不可靠、速率高。

**案例（航空运营智能管理平台）**：对外 **HTTPS 默认 443**（HTTP 80 跳转）；内部 **服务注册发现** 可能用 **53**；**SMTP 25** 若发邮件通知；运维 **SNMP 161/162** 监控网络与健康检查。**TCP** 承载 **强一致票务与事务型 API**；**UDP** 多用于 **DNS、VoIP/部分遥测**（按架构选型）。

## 七、IP 地址与子网（★★★★★）

- **A 类**：首字节 0–127；**B 类**：128–191；**C 类**：192–223；**D 类**：224–239 组播；**E 类**：240–255 保留。
- **网络地址**：主机位全 0。**广播地址**：主机位全 1。**回环地址**：127.0.0.1。
- **子网**：从主机位借 K 位作子网号，可划分 2^K 个子网。**子网掩码**：网络+子网位为 1，主机位为 0；两 IP 与掩码按位与结果相同则在同子网。
- **CIDR**：无类；格式 A.B.C.D/**前缀长度**；网络地址=IP 与掩码按位与。最长前缀匹配：选掩码位数最多的匹配项。

**案例（航空运营智能管理平台）**：云上 **VPC 划分子网**（Web 区、应用区、数据区、大数据区），**安全组/ACL** 按 **CIDR** 放通；**K8s Service/Pod CIDR** 与业务网 **最长前缀匹配** 决定路由；**白名单对接民航局/支付** 常配置对端 **固定网段**。

## 八、IPv4 vs IPv6

- IPv4：32 位、点分十进制；IPv6：128 位、冒号十六进制。
- IPv6：固定 40 字节报头、无校验和、内置 IPsec、无状态地址自动配置（SLAAC）、无需 NAT。

**案例（航空运营智能管理平台）**：公网仍以 **IPv4 + NAT** 为主时，考点记 **IPv6 特征** 即可；大型云与 **双栈接入** 可缓解地址枯竭，与 **多终端、海量连接** 长期相关。

## 九、局域网拓扑与以太网（★★★★★）

| 拓扑 | 特点 |
|------|------|
| **星形** | 以枢纽为中心，依赖枢纽 |
| **总线** | 共享线路，线路故障影响全网 |
| **环形** | 闭合环，单点故障影响环路 |
| **网状** | 任意两点直连，可靠但成本高 |

**以太网**：CSMA/CD（载波监听、冲突检测）；监听算法：非坚持、1-坚持、P-坚持；冲突时边发边听、停发、发拥塞信号、随机退避重试。

**案例（航空运营智能管理平台）**：数据中心 **叶脊（Spine‑Leaf）** 多为 **星形扩展的交换结构**；核心链路 **冗余 mesh 思想** 提升可用性，契合 **99.99%**；传统 **CSMA/CD** 在 **交换式全双工以太网** 中冲突域已缩小，考点保留概念即可。

## 十、无线局域网

- Wi-Fi 4（802.11n）~0.6Gbps；Wi-Fi 5（ac）~6Gbps；Wi-Fi 6（ax）~9.6Gbps；Wi-Fi 7（be）~46Gbps。

**案例（航空运营智能管理平台）**：素材 **多终端访问**——旅客 **4G/5G + Wi‑Fi 6/7** 访问购票与行程；航站/客服现场终端同样依赖无线质量；弱网下的 **重试、幂等与超时** 属于应用层配合传输体验。

## 十一、网络设备与交换（★★★★★）

| 设备 | 工作层次 | 功能 |
|------|----------|------|
| 中继器、集线器 | **物理层** | 再生信号；集线器=多口中继器 |
| 网桥、二层交换机 | **数据链路层** | 按 MAC 转发；交换机=多口网桥 |
| 路由器、三层交换机 | **网络层** | 按 IP 路由；三层交换机=带路由的二层交换 |
| 网关 | 高层（4–7） | 不同协议子网互连 |

**交换**：**电路交换**（独占通路）；**报文交换**（存储转发）；**分组交换**（数据报/虚电路，最常用）。

**路由协议**：**内部网关** RIP、OSPF 等；**外部网关** BGP；距离向量 vs 链路状态。

**案例（航空运营智能管理平台）**：

- **物理层**：光模块、光纤、**负载均衡硬件**（部分）。
- **二层**：**TOR 交换机**、**VPC 内二层隔离**。
- **三层**：**边界路由器、三层交换机**、**云厂商 VPC 路由表**；多 AZ **健康检查与漂移**。
- **网关/七层**：**API 网关、WAF、TLS 终结**——素材中 **防 DDoS、TLS、南北向流量**。
- **交换方式**：互联网业务以 **分组交换** 为主；**专线** 可类比 **电路交换** 保障稳定时延。
- **BGP**：**多云/多地域** 与 **公网 Anycast**（若采用）与外部路由策略相关；IDC **OSPF** 为常见内部路由。

## 十二、速记清单（考前扫一眼）

1. **传输**：单工/半双工/全双工。
2. **奈氏**：2W×log2(V)；**香农**：W×log2(1+S/N)。
3. **OSI**：物联网书会使用；**TCP/IP**：应传网接；FTP 20/21、HTTP 80、DNS 53、DHCP 67/68。
4. **IP**：A 0–127、B 128–191、C 192–223；网络地址主机位全 0、广播全 1；同子网=与掩码与后相等。
5. **CIDR**：最长前缀匹配。
6. **拓扑**：星/总/环/网；以太网 CSMA/CD。
7. **设备**：中继器/集线器物理层；网桥/二层交换数据链路；路由器/三层交换网络层；网关高层。
8. **交换**：电路/报文/分组；路由：内部 RIP/OSPF、外部 BGP。
