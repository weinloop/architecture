# 技术英语入门 · 计算机常用词汇

> 对应计划：**Day 1** · 技术英语入门，计算机常用词汇  
> 每词附 **2 条工作日常例句**，便于在会议、文档、沟通中直接使用。

---

## 一、架构与系统

| 单词/词组 | 释义 | 工作日常例句 |
|-----------|------|----------------|
| **architecture** | 架构 | ① We need to review the **architecture** before the next sprint. ② The **architecture** doc is in Confluence under "Tech Design". |
| **scalability** | 可扩展性 | ① **Scalability** is a key requirement for this service. ② We're discussing **scalability** in the design review tomorrow. |
| **deployment** | 部署 | ① The **deployment** to production is scheduled for Friday. ② We use CI/CD for automated **deployment**. |
| **latency** | 延迟 | ① We need to reduce **latency** for the API response. ② The **latency** is under 100ms in most regions. |
| **throughput** | 吞吐量 | ① The new version improves **throughput** by about 30%. ② We're measuring **throughput** under load. |

## 二、开发与接口

| 单词/词组 | 释义 | 工作日常例句 |
|-----------|------|----------------|
| **API** | 应用程序接口 | ① Please check the **API** docs before integrating. ② We'll expose an **API** for the order service. |
| **endpoint** | 接口/端点 | ① The **endpoint** returns JSON. ② Add the new **endpoint** to the API gateway. |
| **backend** | 后端 | ① The **backend** team will release the fix today. ② I'm mainly responsible for **backend** development. |
| **frontend** | 前端 | ① **Frontend** and backend need to agree on the contract. ② The **frontend** is built with React. |
| **integration** | 集成 | ① The **integration** with the payment system is done. ② We're doing **integration** testing this week. |

## 三、数据与存储

| 单词/词组 | 释义 | 工作日常例句 |
|-----------|------|----------------|
| **database** | 数据库 | ① The **database** migration runs at midnight. ② We're moving the **database** to the new cluster. |
| **cache** | 缓存 | ① We use Redis as **cache** to reduce DB load. ② Clear the **cache** after you deploy the config change. |
| **replica** | 副本 | ① Read traffic goes to the **replica**. ② We have one primary and two **replicas**. |
| **query** | 查询 | ① This **query** is too slow; we need an index. ② Run the **query** in the staging DB first. |
| **transaction** | 事务 | ① Wrap these updates in a **transaction**. ② The **transaction** failed and was rolled back. |

## 四、运维与监控

| 单词/词组 | 释义 | 工作日常例句 |
|-----------|------|----------------|
| **monitoring** | 监控 | ① **Monitoring** shows no errors in the last hour. ② We set up **monitoring** for the new service. |
| **log** | 日志 | ① Check the **log** for the exact error message. ② **Logs** are aggregated in ELK. |
| **alert** | 告警 | ① We got an **alert** for high CPU usage. ② Configure the **alert** to notify the on-call engineer. |
| **rollback** | 回滚 | ① We had to **rollback** the release due to a bug. ② The **rollback** script is in the runbook. |
| **downtime** | 停机时间 | ① Plan the migration during low **downtime** window. ② We had 5 minutes of **downtime** last night. |

## 五、流程与协作

| 单词/词组 | 释义 | 工作日常例句 |
|-----------|------|----------------|
| **sprint** | 迭代 | ① This story will be done in the next **sprint**. ② We're in **sprint** planning right now. |
| **release** | 发布 | ① The **release** is blocked by a failing test. ② We **release** to production every two weeks. |
| **review** | 评审 | ① Can you **review** my PR when you have time? ② The design **review** is at 3pm. |
| **ticket** | 工单/任务 | ① I'll create a **ticket** for this bug. ② The **ticket** is assigned to the platform team. |
| **deadline** | 截止日期 | ① The **deadline** for the POC is next Monday. ② We're pushing to meet the **deadline**. |

## 六、常用动词/短语

| 单词/词组 | 释义 | 工作日常例句 |
|-----------|------|----------------|
| **deploy** | 部署 | ① We **deploy** to staging first, then production. ② I'll **deploy** the fix after the review. |
| **debug** | 调试 | ① I'm **debugging** the issue in the payment flow. ② We need more logs to **debug** this. |
| **refactor** | 重构 | ① We plan to **refactor** this module in Q2. ② **Refactor** the code before adding new features. |
| **reproduce** | 复现 | ① I couldn't **reproduce** the bug locally. ② Can you help **reproduce** the issue on your machine? |
| **onboard** | 入职/接入 | ① I'll **onboard** the new service to our monitoring. ② The **onboarding** doc is in the wiki. |

## 使用建议

- **早上 30 分钟**：选 5–8 个词，朗读例句并尝试替换主语/场景自己造句。
- **工作中**：在写英文邮件、看文档、开会时留意这些词，遇到就记一笔。
- **复习**：周末把本周勾选的词再过一遍，用 1 句话描述「上周我在哪用到这个词」。
