# 类图 · 六种关系与强弱

**知识点编号**：2.4.3.3  
**权重**：超级重点 ★★★★★

## 强弱顺序（必背）

**泛化 = 实现 ＞ 组合 ＞ 聚合 ＞ 关联 ＞ 依赖**

**口诀**：饭（泛）食（实现）组聚关羽（依）。

## 六种关系要点

| 关系 | 要点 |
|------|------|
| **关联** | 稳定结构联系；可多重性、角色名 |
| **依赖** | 临时使用；一方变化可能影响另一方 |
| **泛化** | 父—子继承，is-a |
| **实现** | 类实现接口 |
| **聚合** | 整体—部分；部分可属多个整体，**生命周期独立** |
| **组合** | 整体—部分；部分通常只属于一整体，**生命周期一致** |

- **聚合/组合** 都是 **关联的特化**。  
- **CRC**：类—责任—协作者（定义概念类、关系、职责等步骤的统称）。

## 考点提示

聚合 vs 组合（生命周期）；依赖 vs 关联（稳定 vs 临时）。
