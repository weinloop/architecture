# 影响软件可靠性的五大因素

**知识点编号**：7.3  
**权重**：非重点（概念选择）

## 核心要点

**口诀**：**还闺内发钱** — **环**境（运行剖面）、**规**模、**内**部结构、**发**（开发方法与工具环境）、**钱**（可靠性投入）。

| 因素 | 说明 |
|------|------|
| **运行剖面（环境）** | 平台、硬件、用户习惯、输入特征等 |
| **软件规模** | 代码量、模块数、功能面 |
| **内部结构** | 耦合、内聚、复杂度、层次 |
| **开发方法与环境** | 过程规范、CI/CD、静态分析等 |
| **可靠性投入** | 测试、验证、评审、容错、监控等资源 |
