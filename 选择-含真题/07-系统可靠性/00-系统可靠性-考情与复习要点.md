# 系统可靠性 · 考情与复习要点

**章节权重**：次重点 ★★★☆☆（选择题为重；案例少；论文偏理论与可靠性评价/设计）

## 核心要点

- **定义**：在规定**条件**与**时间区间**内完成规定功能的能力。
- **必背**：**MTBF / MTTR / MTTF / MTTA / MTTD** 含义与后缀联想；**可靠度 R(t)**、**失效概率 F(t)**，**R(t)=1−F(t)**；常考 **MTTR 很小时 MTBF≈MTTF**。
- **计算**：**McCabe** V(G)；**串联**可靠度连乘、**并联** R=1−∏(1−Ri)。
- **设计**（论文可展开）：恢复块、**N 版本**、冗余、检错、降复杂度、双机/集群等。
