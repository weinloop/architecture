# 处理流程设计 · 流程要素与工具

**知识点编号**：3.3～3.3.2  
**权重**：次重点 ★★★☆☆

## 核心要点

### 流程相关概念

- **流程（ISO 9000）**：将输入转化为输出的相互关联或相互作用的活动。  
- **六要素口诀**：**输入输出活动（相互作用）用户价值**。  
- **工作流（WfMC）**：可部分或全自动执行的业务过程，按规则在参与者间传递文档/任务。  
- **活动 / 所有者 / 工作项**：活动改变数据或状态；所有者有权结束活动；工作项为实例中待办。

### 流程设计工具（三类）

| 大类 | 工具 | 要点 |
|------|------|------|
| **图形** | 程序流程图 PFD | 控制流直观；箭头随意转移需约束 |
| | N-S 图（盒图） | 无箭头、强结构化；顺序/选择/循环等 |
| | **IPO 图** | **输入-处理-输出**；加工说明可配流程图、判定树/表 |
| | PAD | 五种基本结构，可递归；日立提出 |
| | 判定树 | 多条件分支 |
| **表格** | 判定表 | 条件与动作矩阵 |
| **语言** | PDL（伪代码） | 自然语言 + 结构化语法 |

**辨析**：**PFD 描述控制流**；描述「模块的输入、输出与加工」更典型的是 **IPO 图**。

### 结构化程序三种基本控制结构

**顺序、分支（选择）、循环** — 可构造单入口单出口程序。

## 系统架构设计师 · 真题汇编

### （2015年11月·第22题）
关于处理流程设计工具，不正确的是（）。  
**答案**：A（「PFD 用于描述各模块的输入、输出和数据加工」）  
**要点**：该描述对应 **IPO**；PFD 侧重 **控制流**。

### （2021年11月·第24题）
不属于**结构化设计**工具的是（）。  
**答案**：C **顺序图**（UML/OO）  
**要点**：盒图、HIPO、程序流程图属结构化阵营。

### （2021年11月·第26题）【McCabe】
用 McCabe 方法计算程序流程图环形复杂度，图中有 3 个判定节点，则 V(G)=（）。  
**答案**：B **4**（V(G)=P+1）  
**要点**：属详细/复杂度度量，常与流程图同卷出现。

### （2017年11月·第24题）
结构化程序设计三种基本控制结构是（）。  
**答案**：B **顺序、分支和循环**  
**要点**：非「嵌套」「并发」「goto」。
