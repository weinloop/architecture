# 运维技术指标 · MTTR / MTBF / MTTF / MTTA

**知识点编号**：6.1  
**权重**：次重点 ★★★☆☆

## 核心要点

量化运维能力，与**可靠性**章节指标一致，需掌握**含义 + 计算式**。

| 指标 | 含义 | 计算 |
|------|------|------|
| **MTTR** 平均故障修复时间 | 从故障发生到恢复正常平均耗时（含通知、诊断、修复、验证） | 时间段内修复总时间 / 维修次数 |
| **MTBF** 平均故障间隔时间 | 两次故障之间的平均运行时间（正常→下次故障） | 故障间总运行时间 / 故障总数 |
| **MTTF** 平均无故障时间 | 不可修或部件级：正常功能持续时间的平均 | 周期内正常运行总时间 / 故障总数 |
| **MTTA** 平均应答时间 | 告警出现到**确认**的平均时间（运维响应） | 告警到确认累计时间 / 事件总数 |

### （2021年11月·第37题）

在 Web 服务器的测试中，反映其性能的指标不包括:（），常见的 Web 服务器性能评测方法有基准
性能测试、压力测试和 （）。
问题（1）
（A）链接正确跳转
（B）最大并发连接数
（C）响应延迟
（D）吞吐量
问题（2）
（A）功能测试
（B）黑盒测试
（C）白盒测试
（D）可靠性测试

**正确答案：** A,D

**解析**

本题考察的是Web 服务器性能测试的核心指标与常见测试方法。
 Web 服务器的性能测试旨在评估其在不同负载情况下的服务能力和稳定性，主要关注响应速度、处
理能力和系统资源利用等方面。
问题1：
A选项 链接正确跳转：正确。链接跳转的正确性属于功能测试范畴，不是衡量服务器性能的指标
，因此本项不属于性能指标。
B选项 最大并发连接数：错误。是衡量服务器在同一时刻能处理多少客户端连接的关键性能指标。
C选项 响应延迟：错误。指服务器响应客户端请求所需的时间，是基本性能指标之一。
D选项 吞吐量：错误。表示单位时间内服务器能处理的请求数量，是服务器处理能力的直接体现。
 所以选择 A。
问题2：
A选项 功能测试：错误。功能测试关注系统功能是否正确实现，非性能评估方法。
B选项 黑盒测试：错误。是软件测试的一种方法，测试外部行为，不属于性能测试分类。
C选项 白盒测试：错误。也是一种测试技术，关注内部逻辑实现，不属于性能测试分类。
D选项 可靠性测试：正确。可靠性测试是评估系统在长时间运行过程中稳定性和故障容忍能力的测

试方式，是 Web 性能测试中常见的补充手段。
 所以选择 D。

### （2018年11月·第14题）

为了优化系统的性能，有时需要对系统进行调整。对于不同的系统，其调整参数也不尽相同。例如
，对于数据库系统，主要包括 CPU /内存使用状况、（）、进程/线程状态、日志文件大小等。对于
应用系统，主要包括应用系统的可用性、响应时间、（）、特定应用的系统资源占用等。
问题（1）
（A）数据丢包率
（B）端口吞吐量
（C）数据处理速率
（D）查询语句性能
问题（2）
（A）并发用户数
（B）支持协议和标准
（C）最大连接数
（D）时延抖动

**正确答案：** D,A

**解析**

本题考察的是数据库系统性能优化的关键参数。
 问题 1：数据库系统的性能调整通常涉及：CPU/内存使用状况、查询语句性能、进程/线程状态、
日志文件大小等因素。
A选项数据丢包率：这是网络传输中的指标，主要反映网络层面问题，和数据库系统本身性能关联不
大。
B选项端口吞吐量：也是网络相关的参数，关注的是数据进出速率，而不是数据库内部的处理能力。
C选项数据处理速率：虽然也是性能指标，但过于笼统，不如“查询语句性能”直接、准确。
D选项查询语句性能：数据库中大量操作涉及查询，查询语句设计优劣直接决定了系统的响应效率和
资源消耗，因此是关键优化对象。
 所以，正确答案是 D。
 问题 2：应用系统的性能调整通常关注：可用性、响应时间、并发用户数、系统资源占用等方面。
A选项并发用户数：反映系统同时处理多少用户请求，是衡量应用系统性能的核心指标之一。
B选项支持协议和标准：属于系统设计或兼容性范畴，与运行时性能无关。
C选项最大连接数：虽然与系统负载能力有关，但它是系统设定的上限，并不能直接反映当前系统性
能表现。
D选项时延抖动：这是网络通信中的指标，关注的是数据包延迟的稳定性，对应用系统的性能评估价
值有限。
 所以，正确答案是 A。

### （2017年11月·第13题）

对计算机评价的主要性能指标有时钟频率、（）、运算精度和内存容量等。对数据库管理系统评价
的主要性能指标有（）、数据库所允许的索引数量和最大并发事务处理能力等。
问题（1）
（A）丢包率
（B）端口吞吐量
（C）可移植性
（D）数据处理速率
问题（2）
（A）MIPS

（B）支持协议和标准
（C）最大连接数
（D）时延抖动

**正确答案：** D,C

**解析**

本题考察的是硬件与软件系统的主要性能指标。
问题 1：
 计算机的主要性能指标除了时钟频率外，还包括数据处理速率（即运算速度）、运算精度和内存容
量等。
A选项丢包率：这是网络设备（如路由器、交换机）的性能指标，不是计算机本身的主要性能指标
，错误。
B选项端口吞吐量：主要用于描述网络设备端口的数据转发能力，与计算机整体性能无直接对应，错
误。
C选项可移植性：这是软件质量特性，不是计算机硬件性能指标，错误。
D选项数据处理速率：反映了计算机处理数据的能力，是硬件重要性能指标，正确。
 选择选项 D。
问题 2：
 数据库管理系统（DBMS）的性能指标包括最大连接数、允许的索引数量、最大并发事务能力等。
A选项MIPS：表示每秒执行的百万条指令数，是计算机处理器的性能指标，不是DBMS特有的指标，错
误。
B选项支持协议和标准：这是系统兼容性和可移植性方面的指标，不是数据库性能核心指标，错误。
C选项最大连接数：表示数据库同时支持的最大连接用户数量，是重要的性能指标之一，正确。
D选项时延抖动：是网络通信领域的指标，与数据库性能无直接关联，错误。
 选择选项 C。
