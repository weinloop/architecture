## 核心要点（背诵）

- **SoC**：片上系统，多将 CPU、存储控制器、各类 IP、外设等集成于单芯片，面向特定用途；不等于「只是一块处理器芯片」。
- **嵌入式处理器分类**：MPU、MCU（单片机）、DSP、SoC；**DSP** 多采用 **哈佛结构**（指令与数据存储分离、可并行访问），通用 CPU 多为 **冯·诺依曼**。
- **MMU**：无 MMU 的典型如 **Cortex-M** 系列，常配 **μC/OS-II** 等轻量 RTOS；需 MMU 的如 Linux、Windows CE 等。
- **AI / NPU / GPU**：AI 芯片强调 **训练与推理**、新计算范式、大数据与可重构等；**NPU** 侧重神经网络加速、能效；**GPU** 峰值算力常以 **TFLOPS** 计。
- **嵌入式软件架构**（真题表述）：常见概括为 **层次化** 与 **递归** 等模型（与通用分布式「代理/点对点」等区分）。
- **器件等级**：如 **工业级** 温度范围常考 **−40℃～85℃**（与民用 0～70℃、军用更宽对比）。

### （2022年11月·第11题）

AI芯片是当前人工智能技术发展的核心技术，其能力要支持训练和推理。通常，AI芯片的技术架构
包括（）等三种。
问题（1）
（A）GPU、FPGA、ASIC
（B）CPU、PPGA、DSP
（C）GPU、CPU、ASIC
（D）GPU、FPGA、SOC

**正确答案：** A

**解析**

本题考察的是AI芯片的基本技术架构的相关概念。
 AI芯片需要在训练和推理两个阶段提供高效的计算能力，而GPU、FPGA和ASIC是当前主流的三种技
术路线。
A选项GPU、FPGA、ASIC：GPU擅长大规模并行计算，是深度学习训练的重要硬件平台；FPGA支持硬件
可编程，适用于灵活部署和低功耗场景；ASIC为特定AI任务定制，性能和能效比极高，适合大规模
推理部署，三者确为当前主要架构。
B选项CPU、PPGA、DSP：CPU虽可参与AI计算，但并非主流高性能AI芯片架构；“PPGA”应为
“FPGA”，这里拼写错误；DSP主要用于信号处理，不是AI芯片的三大主流架构。
C选项GPU、CPU、ASIC：虽然GPU和ASIC正确，但CPU并不是高效支持AI大规模计算的核心芯片类型。
D选项GPU、FPGA、SOC：SOC是一种系统级芯片设计形式，并不是AI芯片技术架构的特定分类。
 因此，选项A正确。

### （2022年11月·第10题）

GPU 目前已广泛应用于各行各业。最新的GPU峰值性能可高达（）以上。
问题（1）
（A）100 TFlops
（B）50 TFlops
（C）10 TFlops
（D）1 TFlops

**正确答案：** A

**解析**

本题考察的是GPU峰值性能与 FLOPS 单位的相关概念。
A选项 100 TFlops：FLOPS（Floating Point Operations Per Second）表示每秒浮点运算次数
，TFlops 表示万亿次浮点运算。最新的高端 GPU（如 NVIDIA RTX 4090、H100 等）单精度浮点运
算峰值性能已超过 100 TFlops，因此该选项正确。
B选项 50 TFlops：虽然中高端 GPU 的性能在 50 TFlops 左右，但并非最新峰值性能的上限，数值
偏低。
C选项 10 TFlops：这是几年前高性能显卡的水平，现已被大幅超越。
D选项 1 TFlops：属于早期 GPU 性能或低端 GPU 的范围，与当前高性能 GPU 相差甚远。
 因此，选项 A 正确。

### （2021年11月·第7题）

一般说来 SoC 称为系统级芯片，也称片上系统，它是一个有专用目标的集成电路产品，以下关于
SoC 不正确的说法是（）
问题（1）
（A）SoC 是一种技术，是以实际的、确定的系统功能开始，到软/硬件划分，并完成设计的整个过
程
（B）SoC 是一款具有运算能力的处理器芯片，可面向特定用途进行定制的标准产品
（C）SoC 是信息系统核心的芯片集成，是将系统关键部件集成在一块芯片上，完成信息系统的核心
功能
（D）SoC 是将微处理器、模拟IP 核、数字 IP 核和存储器(或片外存储控制接口)集成在单一芯片
上，是面向特定用途的标准产品

**正确答案：** B

**解析**

本题考察的是系统级芯片（SoC）的定义与特征。
 SoC（System on Chip）是一种面向特定应用的集成电路，将系统的关键功能部件集成在单一芯片
上，通常包括微处理器、模拟模块、数字模块和存储器等，并配合软件实现完整的系统功能。
A选项：正确，SoC 的设计过程通常从特定系统功能出发，经过软硬件划分，再完成整体设计，符合
定义。
B选项：错误，将 SoC 直接等同于一块“处理器芯片”是片面的，SoC 并不仅仅是 CPU，而是一个
包含处理器、外设、存储等在内的完整系统。
C选项：正确，从狭义角度看，SoC 是将信息系统的关键部件集成到单芯片上的核心硬件，实现系统
的主要功能。
D选项：正确，从学术界通用定义看，SoC 将微处理器、模拟 IP 核、数字 IP 核及存储器集成在同
一芯片上，通常是面向特定用途的标准或定制产品。
 因此，选项 B 正确。

### （2021年11月·第3题）

人工智能技术已成为当前国际科技竞争的核心技术之一，AI 芯片是占据人工智能市场的法宝。AI
芯片有别于通常处理器芯片，它应具备四种关键特征。（）是 AI 芯片的关键特点。
问题（1）
（A）新型的计算范式、信号处理能力、低精度设计、专用开发工具
（B）新型的计算范式、训练和推断、大数据处理能力、可重构的能力
（C）训练和推断、大数据处理能力、可定制性，专用开发工具
（D）训练和推断、低精度设计、新型的计算范式、图像处理能力

**正确答案：** B

**解析**

本题考察的是AI 芯片的关键特征。
A选项：包含信号处理能力和低精度设计，这些虽然可能是 AI 芯片设计的部分特点，但不是官方归
纳的四大关键特征，错误。
B选项：新型的计算范式（如神经网络计算）、支持训练和推断、大数据处理能力、可重构能力，完
全符合 AI 芯片的核心特性，正确。
C选项：可定制性与专用开发工具并非 AI 芯片四大关键特征的标准表述，错误。
D选项：低精度设计和图像处理能力虽然在部分 AI 芯片中常见，但不在官方归纳的四大特征中，错
误。
 因此选择 B。

### （2018年11月·第7题）

目前处理器市场中存在CPU和DSP两种类型处理器，分别用于不同场景，这两种处理器具有不同的体
系结构，DSP采用（）。
问题（1）
（A）冯诺伊曼结构
（B）哈佛结构
（C）FPGA结构
（D）与GPU相同结构

**正确答案：** B

**解析**

本题考察的是DSP芯片的体系结构特点。
 DSP（Digital Signal Processor）是一种专门用于数字信号处理的微处理器，其设计理念是为了
满足高速、实时处理的需求。
A选项 冯诺伊曼结构：程序与数据共享同一存储空间与总线，指令和数据不能同时访问，限制了速
度，不是DSP的主流结构，错误。
B选项 哈佛结构：指令存储器与数据存储器分开，程序和数据可以同时访问，大幅提升处理速度
，是DSP采用的经典体系结构，正确。
C选项 FPGA结构：FPGA是一种可编程逻辑器件，不是处理器体系结构，错误。
D选项 与GPU相同结构：GPU是大规模并行处理架构，设计目的不同，与DSP体系结构不相同，错误。
 因此，本题答案为 B。

### （2016年11月·第3题）

嵌入式处理器是嵌入式系统的核心部件，一般可分为嵌入式微处理器（MPU）、微控制器（MCU）、
数字信号处理器（DSP）和片上系统（SOC）。以下叙述中，错误的是（）。
问题（1）
（A）MPU在安全性和可靠性等方面进行增强，适用于运算量较大的智能系统
（B）MCU典型代表是单片机，体积小从而使功耗和成本下降
（C）DSP处理器对系统结构和指令进行了特殊设计，适合数字信号处理
（D）SOC是一个有专用目标的集成电路，其中包括完整系统并有嵌入式软件的全部内容

**正确答案：** A

**解析**

本题考察的是嵌入式处理器的分类与特点。
A选项MPU：MPU是增强型通用微处理器，确实在工作温度、电磁兼容性、可靠性等方面会有提高，但
功能上与通用微处理器基本一致，并非专为“运算量较大的智能系统”而设计，这种表述容易与高
性能处理器混淆，因此该项表述错误。
B选项MCU：微控制器（单片机）将CPU、RAM、ROM、定时器、I/O接口等集成在一片芯片上，体积小
、功耗低、成本低，适合控制类应用，该描述正确。
C选项DSP：数字信号处理器在硬件结构和指令集上进行优化，具备快速乘法器、流水线结构、特殊
寻址模式等，非常适合进行实时数字信号处理，该描述正确。
D选项SoC：片上系统将CPU、存储器、I/O接口、专用功能模块等集成在一块芯片上，形成完整的系
统，并可包含嵌入式软件，该描述正确。
 因此，答案是 A。

### （2014年11月·第6题）

以下嵌入式处理器类型中不具备内存管理单元（MMU）的是（），嵌入式操作系统（）可以运行在它
上面。
问题（1）
（A）PowerPC750
（B）ARM920T
（C）Cortex-M3
（D）MIPS32 24K
问题（2）
（A）Linux
（B）VxWorks653
（C）UC/OS-II
（D）Windows CE

**正确答案：** C,C

**解析**

本题考察的是嵌入式处理器架构与嵌入式操作系统的适配关系。
问题1：
A选项 PowerPC750：高性能处理器，具备 MMU，能支持完整的 Linux 或 Windows CE。
B选项 ARM920T：属于 ARM9 系列，具备 MMU，可运行嵌入式 Linux。
C选项 Cortex-M3：定位于低功耗、实时控制应用，不具备 MMU，仅有 MPU（内存保护单元），因此
正确。
D选项 MIPS32 24K：该系列包含 MMU，可运行 Linux。
 因此，选择选项 C。
问题2：
A选项 Linux：需要 MMU 支持，不能在 Cortex-M3 上运行。
B选项 VxWorks653：通常运行在具备 MMU 的高端处理器上，适用于航空航天等实时应用。
C选项 UC/OS-II：轻量级嵌入式实时操作系统，不依赖 MMU，可以运行在 Cortex-M3 上，正确。
D选项 Windows CE：需要 MMU 支持，不能运行在 Cortex-M3 上。
 因此，选择选项 C。

### （2023年11月·第38题）

关于 NPU（神经网络处理器）的说法，正确的是（）。
问题（1）
（A）计算能力强，但能耗高
（B）计算速度快，性能好
（C）应用广泛，但成本高
（D）设计复杂，性能一般

**正确答案：** B

**解析**

本题考察的是NPU（神经网络处理器）的特点。
 NPU 是专门为神经网络计算优化的处理器，能够通过硬件级优化和专用指令集，大幅提升深度学习
中的矩阵运算效率，常用于推理和训练加速。
A选项计算能力强，但能耗高：NPU 的设计目标就是在保证高性能的同时尽可能降低能耗，与 GPU
相比其能效比更高，因此“能耗高”是错误的。
B选项计算速度快，性能好：NPU 的最大优势就是计算速度快、并行能力强、功耗低，在 AI 场景下
性能远优于 CPU 和 GPU，因此这是正确的。
C选项应用广泛，但成本高：目前 NPU 还处于快速发展阶段，应用主要集中在 AI 相关领域（如智
能手机、自动驾驶、云计算加速），尚不能称为“应用广泛”；同时成本相较于传统 CPU、GPU 不
一定更高，因此该表述不准确。
D选项设计复杂，性能一般：NPU 的确在架构设计上复杂，但其性能并不是“一般”，而是其突出优
势，因此该说法错误。
 因此，正确答案是 B。

### （2024年5月·第12题）

大多数嵌入式系统都具备实时特征，其典型架构可概括为（）两种模型。
问题（1）
（A）层次化模式架构和代理模式架构
（B）层次化模式架构和点对点模式架构
（C）层次化模式架构和递归模式架构
（D）递归模式架构和点对点模式架构

**正确答案：** C

**解析**

本题考察的是嵌入式系统的软件架构设计模型。
 嵌入式系统由于其专用性和实时性的特点，其架构设计必须根据具体应用目标进行定制，常见的设
计模式包括层次化模式架构和递归模式架构。
层次化模式架构：将系统按功能划分为多个层次，各层之间具有明确的依赖关系与调用关系。上层
依赖下层服务，而下层对上层无感知，具有良好的可扩展性和可维护性。
递归模式架构：适用于嵌套性强、资源重复使用的系统结构，它强调组件之间的递归调用，通常用
于操作系统内核、调度机制等设计中。
A选项代理模式架构：主要用于分布式系统中的远程通信，不属于嵌入式系统的典型架构。
B选项点对点模式架构：用于通信或网络系统的连接方式，也不是嵌入式系统架构的主流形式。
D选项排除理由同上。
 因此，选项 C 正确。

### （2025年5月·第11题）

某芯片的工作温度范围为 -40°C 到 85°C，根据芯片的使用环境等级分类，该芯片应属于（）。
问题（1）
（A）民用级
（B）工业级
（C）通用级
（D）军用级

**正确答案：** B

**解析**

芯片按其工作环境温度范围一般分为以下几类：
民用级（Commercial Grade）：工作温度范围为 0°C 至 70°C，适用于普通消费电子产品。
工业级（Industrial Grade）：工作温度范围为 -40°C 至 85°C，适用于工业自动化、嵌入式设
备等需要适应较宽温度范围的场景。
军用级（Military Grade）：工作温度范围为 -55°C 至 125°C 或更宽，适用于航空航天、军事
装备等高可靠性要求场合。
通用级：该选项并不是标准分类术语，干扰项。
 所以选 B。
