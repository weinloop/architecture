9.3.输入输出系统（次重点★★★☆☆)


| 在计算机中 ，I/O  系统可以有 5 种不同的工作方式 ，分别是程序控制方式、程序中断方式、 DMA  工作方式、通道方式、I/O  处理机。你只需要知道 DMA 是主存和外设之间交换数据即可。 |
| --- |



| 数据传送方式 | 控制主体 | 工作原理 | 特点 |
| --- | --- | --- | --- |
| 程序控制方式 | CPU | CPU  执行  I/O  程序实现数据传送，分无条件传送和程序查询方式 | 简单 ，CPU  参与度高 ，效率低 |
| 程序中断方式 | CPU | CPU  执行现行程序时，遇异常或特殊请求 ，暂时中止程序处理紧急事件 ，处理后返回原程序 | 提高 I/O  能力和 CPU  效率，CPU需处理中断 |
| DMA 工作方式 | DMA  控制器 | 在主存与外设间实现高速、批量数据交换 ，DMAC  控制管理数据传输 | 高速批量传输 ，减少 CPU  干预 |
| 通道方式 | 通道 | 在一定硬件基础上利用软件手段实现I/O  控制和传送 | 主机和外设并行程度高，减少 CPU介入 |
| I/O  处理机 | I/O    处 理 机（ 专 用 或 通用 ） | 具有丰富指令系统和完善中断系统，独立处理 I/O  操作 | 高度自治，分担 CPU I/O  处理任务 |

9.3.1.总线（次重点★★★☆☆)


| 总线是一组能为多个部件分时共享的公共信息传送线路。共享是指总线上可以挂接多个部件， |
| --- |



| 各个部件之间相互交换的信息都可以通过这组公共线路传送。分时是指同一时刻只允许有一个部件向总线发送信息 ，如果出现两个或两个以上部件同时向总线发送信息 ，势必导致信号冲突。 当然，在同一时刻 ，允许多个部件同时从总线上接收相同的信息。此章节不重要 ，真题会做即可。 |
| --- |


（ 1 ）总线宽度。总线宽度指的是总线的线数 ，它决定了总线所占的物理空间和成本。例如，32 位的 PCI 总线允许寻址的主存空间的大小为232=4G 个单元。

（2）总线带宽。 总线带宽定义为总线的最大数据传输速率 ， 即每秒传输的字节。

总线带宽 =  总线宽度× 总线频率

例题：假设某系统总线在一个总线周期中并行传输 4B 信息 ，一个总线周期占用2 个时钟周期 ，总线时钟频率为 10MHz  ，则总线的频率为 10MHz/2=5Mhz。

这里要注意的是它们之间是除法。你可以这么理解 ， 时钟信号在一秒钟内周期性变化 10M 次 ，其中 2 次用于总线的传输，那么一秒周期内总线变化或者说可以传输 5M 次 ，总线周期是 5MHz ，单次传输数据量（ 总线宽度为4B ），则总线带宽为4B*5MHz=20MBps。

## 系统架构设计师 · 真题汇编

> 来源：[计算机组成原理.pdf](../../选择真题-分类/计算机组成原理.pdf)。**完整题干、选项、正确答案与解析**。

### （2018年11月·第8题）

以下关于串行总线的说法中，正确的是（）。
问题（1）
（A）串行总线一般都是全双工总线，适宜于长距离传输数据
（B）串行总线传输的波特率是总线初始化时预先定义好的，使用中不可改变
（C）串行总线是按位（bit）传输数据的，其数据的正确性依赖于校验码纠正
（D）串行总线的数据发送和接收是以软件查询方式工作

**正确答案：** C

**解析**

本题考察的是串行总线的基本特性。
 串行总线是指以位为单位在单条数据线上依次传输数据的一种通信方式，常用于长距离通信和数据
量相对较小的场景。
A选项：串行总线既有半双工也有全双工方式，不是“一般都是全双工”，也不适宜于长距离传输数
据，因此错误。
B选项：波特率虽然可以在初始化时设定，但许多串行通信协议支持在使用过程中动态调整波特率
，因此说“不可改变”是错误的。
C选项：串行总线按位传输数据，常配合奇偶校验、CRC等方式进行错误检测与纠正，确保数据正确
性，这一说法正确。
D选项：串行总线的数据传输方式多种多样，可以是软件查询、硬件中断、DMA传输等，不仅限于软
件查询，因此错误。
 因此，本题答案为 C。

### （2017年11月·第2题）

DMA （直接存储器访问）工作方式是在（）之间建立起直接的数据通路。
问题（1）
（A）CPU与外设
（B）CPU与主存
（C）主存与外设
（D）外设与外设

**正确答案：** C

**解析**

本题考察的是直接存储器访问（DMA, Direct Memory Access） 的工作原理。
 DMA 是一种 绕过 CPU、直接在内存和外设之间进行数据成块传输的技术，能够提高数据传输效率
并释放 CPU 处理时间。
A选项 CPU与外设：CPU直接与外设进行数据交换属于程序控制方式，不是DMA，错误。
B选项 CPU与主存：CPU与主存的数据交互属于正常指令执行，不是DMA，错误。
C选项 主存与外设：DMA的核心就是在主存与I/O设备之间直接建立数据通路，由DMA控制器负责数据
传输，正确。
D选项 外设与外设：这种传输通常需要经过主存中转，DMA不直接支持外设对外设的数据传输，错误
。
 因此，本题答案为 C。

### （2015年11月·第9题）

下列说法中正确的是（）。
问题（1）
（A）半双工总线只在一个方向上传输信息，全双工总线可在两个方向上轮流传输信息
（B）半双工总线只在一个方向上传输信息，全双工总线可在两个方向上同时传输信息
（C）半双工总线可在两个方向上轮流传输信息，全双工总线可在两个方向上同时传输信息
（D）半双工总线可在两个方向上同时传输信息，全双工总线可在两个方向上轮流传输信息

**正确答案：** C

**解析**

本题考察的是半双工和全双工总线的传输方式。
A选项：半双工总线并非只在一个方向上传输信息，它可以在两个方向上传输，但不能同时，只能轮
流，因此该说法错误。
B选项：半双工总线并不是单向的，因此说法错误。
C选项：半双工总线可以双向传输，但必须轮流进行；全双工总线可以在两个方向上同时传输信息
，该说法完全正确。
D选项：半双工总线不能在两个方向上同时传输信息，因此说法错误。
 因此，正确答案为 C。

### （2014年11月·第5题）

计算机采用分级存储体系的主要目的是为了（）。
问题（1）
（A）解决主存容量不足的问题
（B）提高存储器读写可靠性
（C）提高外设访问效率
（D）解决存储的容量、价格和速度之间的矛盾

**正确答案：** D

**解析**

本题考察的是计算机分级存储体系（Memory Hierarchy）的基本思想。
A选项：分级存储确实能在一定程度上缓解主存容量不足的问题，但这并非其根本目的，错误。
B选项：读写可靠性主要通过冗余、纠错（ECC）来实现，与分级存储无直接关系，错误。
C选项：外设访问效率的提高依赖缓冲、通道和DMA技术，而不是分级存储体系，错误。
D选项：存储器存在容量、价格、速度三者难以兼顾的矛盾。分级存储体系通过层次化结构（寄存器
→Cache→主存→磁盘）来在速度、价格和容量之间取得平衡，这是采用该体系的主要目的，正确。
 因此，正确答案是 D。

### （2023年11月·第14题）

（）是一种通过执行通道程序管理 I/O 操作的控制器，它使主机（CPU和内存）与 I/O 操作之间达
到更高的并行程度。
问题（1）
（A）系统总线
（B）通道
（C）控制器
（D）存储器

**正确答案：** B

**解析**

本题考察的是通道的概念。
A选项 系统总线：系统总线是连接计算机内部各个部件（如CPU、内存、I/O设备等）的传输通道。
它并不专门负责管理I/O操作的执行，而是用于数据传输，因此与通道的定义不符。
B选项 通道：通道是一种用于管理输入/输出（I/O）操作的控制器，通过执行通道程序来提高CPU和
内存与I/O操作之间的并行性。通道使得I/O操作不需要占用主机（CPU和内存）资源，从而实现了
I/O操作的并行执行，提升了计算机系统的整体性能。它被称为I/O处理机或I/O控制器，是为了减少
CPU与I/O设备之间的直接交互，增加系统的并行性和效率。
C选项 控制器：控制器是硬件部件，它用于控制外设或其他硬件的操作，但并不专门指管理I/O操作
的控制器。虽然控制器在I/O操作中起着重要作用，但它并不是专门为了提高并行度而设计的部件。
D选项 存储器：存储器是用来存储数据和程序的部件，它的作用与I/O操作的并行管理无关，因此这
个选项不正确。
 综上所述，B选项“通道”是正确的。
